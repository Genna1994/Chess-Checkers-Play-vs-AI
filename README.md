**Chess & Checkers vs AI** is an interactive web based board game platform that allows users to play complete matches of both 
Chess and Checkers directly against an intelligent AI opponent. Featuring a clean, responsive layout and fluid piece animations, 
the application delivers a seamless tabletop experience optimized for both desktop and mobile screens. Built with modern 
web technologies, the project showcases robust game logic, turn management, and real time state tracking for an engaging 
single-player experience.
